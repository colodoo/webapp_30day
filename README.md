# webapp_30day  

## 前言  
  
之前写过一个webapp的项目，奈何前端技术渣，只能用UI框架复制粘贴，这次用记录的方式学习一下前端的布局，以及整理一些简单的组件，方便以后使用。  

> 不采用框架方式编写，意在理解各个组件的原理。
